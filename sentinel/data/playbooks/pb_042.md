# PB-042: DNS Tunneling & Data Exfiltration Response

## Playbook Metadata
- **Playbook ID**: PB-042
- **Version**: 1.0
- **Severity**: CRITICAL
- **SLA**: 30 minutes
- **Approval Required**: YES (Security Manager)

## Action Sequence
1. **Identify Source**: Confirm the internal IP and asset (e.g., WKS-RES-042).
2. **Sinkhole Domain**: Add the malicious C2 domain to the DNS sinkhole.
3. **Block IP**: Block the C2 infrastructure IP at the perimeter firewall.
4. **Isolate Asset**: Use EDR to isolate the affected host from the network.
5. **PCAP Capture**: Initiate 30-minute packet capture on the VLAN.
6. **Data Assessment**: Use DLP tools to identify the type and volume of exfiltrated data.

## Containment Logic
If DNS TXT records exceed 500MB/hour, initiate emergency isolation immediately.
