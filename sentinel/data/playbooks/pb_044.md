# PB-044: Phishing Triage and Response

## Playbook Metadata
- **Playbook ID**: PB-044
- **Version**: 1.0
- **Last Updated**: 2026-03-26
- **SLA**: Initial triage within 15 minutes

## Action Sequence

### Step 1 — Analyze Email & URL
- Retrieve raw email headers and body.
- Extract all URLs and attachments.
- **Tool**: Google Threat Intelligence / VirusTotal

### Step 2 — Scope Impact
- Identify all internal recipients of the same email.
- Check proxy logs for other users who visited the malicious domain.

### Step 3 — Containment (If Clicked)
- Reset user's Active Directory password.
- Revoke all active O365/Google Workspace sessions.
- **Tool**: IAM API

### Step 4 — Remediation
- Delete the phishing email from all user mailboxes.
- Block the sender domain at the email gateway.
- Update web proxy blacklist with the malicious URL.

### Step 5 — Notify User
- Send automated notification to the user about the incident and remediation.
