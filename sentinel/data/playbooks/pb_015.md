# PB-015: Compromised Account Response

## Playbook Metadata
- **Playbook ID**: PB-015
- **Version**: 3.2
- **Severity**: MEDIUM
- **SLA**: 60 minutes
- **Approval Required**: Auto-approved for Low/Medium severity.

## Action Sequence
1. **Disable Account**: Disable the user in Active Directory / Azure AD.
2. **Revoke Sessions**: Invalidate all active O365/SAML sessions.
3. **Reset MFA**: Force the user to re-register their MFA device.
4. **Notify User**: Send an automated email to the user (via backup email) and their manager.

## Containment Logic
If login originates from a known Tor exit node or an unapproved country, account suspension is mandatory.
